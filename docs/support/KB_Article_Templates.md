- [Link to related article 2]

## 5. Article Metadata

Every KB article should include:

| Field | Description | Example |
|-------|-------------|---------|
| Title | Clear, searchable | "How to register a custom component" |
| Category | Topic area | "Components" |
| Tags | Keywords | "component, registration, cli" |
| Version | Alloy UI version | "1.0.0" |
| Last Updated | Date | "2025-04-10" |
| Author | Creator name | "Jane Smith" |
| Reviewer | Technical reviewer | "John Doe" |
