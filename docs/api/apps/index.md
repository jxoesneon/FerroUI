# Applications

Deployment targets that consume the FerroUI packages. Each app is a reference implementation — you can copy any of them into your own infrastructure.

| App | Description |
|-----|-------------|
| [`apps/web`](/api/apps/web) | React reference renderer used for Storybook, Playwright E2E, and demos |
| [`apps/edge`](/api/apps/edge) | Cloudflare Worker packaging the engine for edge deployment |
| [`apps/desktop`](/api/apps/desktop) | Tauri-based desktop shell (experimental) |
| [`apps/playground`](/api/apps/playground) | Interactive layout playground (experimental) |
