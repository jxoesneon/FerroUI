# Package Reference

Public API documentation for every package in the FerroUI monorepo. Generated from TypeScript source via TypeDoc on every push.

| [`@ferroui/registry`](/api/packages/registry) | Component registry — atoms, molecules, organisms |
| [`@ferroui/tools`](/api/packages/tools) | Tool registration, permissions, budget enforcement |
| [`@ferroui/i18n`](/api/packages/i18n) | Locale resolution, RTL, translation provider |

| [`@ferroui/renderer`](/api/packages/renderer) | React renderer, action router, state machines |

| [`@ferroui/shared`](/api/packages/shared) | Cross-package internal utilities |
